# Student Task Manager

A small Java console project made for learning Git and GitHub workflow.

## Features

- Add tasks
- Show task list
- Delete tasks
- Mark tasks as completed

## Technologies

- Java
- Git
- GitHub

## Goal

This project is mainly used to practice:
- branches
- merges
- commits
- project organization

## Recent Improvements

- Added task status system
- Improved console interface
- Better project organization
